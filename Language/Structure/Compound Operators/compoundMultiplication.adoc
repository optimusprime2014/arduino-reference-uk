= *= Compound Multiplication (Складене множення)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Це зручне скорочення для виконання множення змінної на іншу константу або змінну.
[%hardbreaks]


[float]
=== Syntax
`x *= y; // еквівалентно виразу x = x * y;`


[float]
=== Parameters
`x`: змінна. Дозволені типи даних: `int`, `float`, `double`, `byte`, `short`, `long`. +
`y`: змінна або константа. Дозволені типи даних: `int`, `float`, `double`, `byte`, `short`, `long`.

--
// OVERVIEW SECTION ENDS



// HOW TO USE SECTION STARTS
[#howtouse]
--

[float]
=== Example Code

[source,arduino]
----
x = 2;
x *= 3; // тепер x містить 6
----
