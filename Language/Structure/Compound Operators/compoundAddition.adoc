= += Compound Addition (Складене додавання)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Це зручне скорочення для виконання додавання значення до змінної або іншої додатної константи чи змінної.
[%hardbreaks]


[float]
=== Syntax
`x += y; // еквівалентно виразу x = x + y;`


[float]
=== Parameters
`x`: змінна. Дозволені типи даних: `int`, `float`, `double`, `byte`, `short`, `long`. +
`y`: змінна або константа. Дозволені типи даних: `int`, `float`, `double`, `byte`, `short`, `long`.

--
// OVERVIEW SECTION ENDS



// HOW TO USE SECTION STARTS
[#howtouse]
--

[float]
=== Example Code

[source,arduino]
----
x = 2;
x += 4; // тепер x містить 6
----
