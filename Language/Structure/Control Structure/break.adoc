= break


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
[%hardbreaks]
`break` використовується для виходу з циклів `link:../for[for]`, `link:../while[while]` або `link:../dowhile[do...while]`, обходячи звичайну умову циклу. Також використовується для виходу з оператора `link:../switchcase[switch case]`.
[%hardbreaks]

--
// OVERVIEW SECTION ENDS




// HOW TO USE SECTION STARTS
[#howtouse]
--
[float]
=== Приклад коду
У наступному коді керування виходить з циклу link:../for[`for`], коли значення датчика перевищує поріг.
[source,arduino]
----
int threshold = 40;
for (int x = 0; x < 255; x++) {
  analogWrite(PWMpin, x);
  sens = analogRead(sensorPin);
  if (sens > threshold) {     // bail out on sensor detect
    x = 0;
    break;
  }
  delay(50);
}
----
