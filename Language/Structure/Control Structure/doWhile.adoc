= do...while loop


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
[%hardbreaks]
Цикл `do...while` працює так само, як і цикл `link:../while[while]`, за винятком того, що умова перевіряється в кінці циклу, тому цикл do завжди виконається принаймні один раз.

[float]
=== Синтаксис
[source,arduino]
----
do {
  // statement block
} while (condition);
----


[float]
=== Параметри
`condition`: логічний вираз, який оцінюється як `link:../../../variables/constants/truefalse[true]` або `link:../../../variables/constants/truefalse[false]`.

--
// OVERVIEW SECTION ENDS




// HOW TO USE SECTION STARTS
[#howtouse]
--

[float]
=== Приклад коду

[source,arduino]
----
// initialize x and i with a value of 0
int x = 0;
int i = 0;

do {
  delay(50);          // wait for sensors to stabilize
  x = readSensors();  // check the sensors
  i++;                // increase i by 1
} while (i < 100);    // repeat 100 times
----
