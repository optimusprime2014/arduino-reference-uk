= && Логічне І (Logical AND)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис (Description)
*Логічне І* (Logical AND) повертає `true` *тільки* якщо обидва операнди є `true`.
[%hardbreaks]

--
// OVERVIEW SECTION ENDS



// HOW TO USE SECTION STARTS
[#howtouse]
--

[float]
=== Приклад коду (Example Code)
Цей оператор можна використовувати всередині умови оператора link:../../control-structure/if[if].

[source,arduino]
----
if (digitalRead(2) == HIGH && digitalRead(3) == HIGH) { // якщо ОБИДВА перемикачі мають значення HIGH
  // твердження (statements)
}
----
[%hardbreaks]

[float]
=== Примітки та попередження (Notes and Warnings)
Переконайтеся, що ви не плутаєте логічний оператор AND, && (подвійний амперсанд) з побітовим оператором AND & (одинарний амперсанд). Це абсолютно різні речі.
