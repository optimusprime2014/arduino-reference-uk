= loop()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Після створення функції `link:../setup[setup()]`, яка ініціалізує та встановлює початкові значення, функція `loop()` робить саме те, що підказує її назва, і послідовно виконує цикл, дозволяючи вашій програмі змінюватися та реагувати. Використовуйте її для активного керування платою Arduino.
[%hardbreaks]

--
// OVERVIEW SECTION ENDS


// HOW TO USE SECTION STARTS
[#howtouse]
--

[float]
=== Приклад коду
[source,arduino]
----
int buttonPin = 3;

// setup ініціалізує послідовний порт та пін кнопки
void setup() {
  Serial.begin(9600);
  pinMode(buttonPin, INPUT);
}

// loop щоразу перевіряє пін кнопки,
// і буде надсилати через послідовний порт, якщо вона натиснута
void loop() {
  if (digitalRead(buttonPin) == HIGH) {
    Serial.write('H');
  }
  else {
    Serial.write('L');
  }

  delay(1000);
}
----
