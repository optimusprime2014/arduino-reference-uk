= word()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Перетворює значення на тип даних `link:../../data-types/word[word]`.
[%hardbreaks]


[float]
=== Syntax
`word(x)` +
`word(h, l)` +
`(word)x` (Перетворення типу у стилі C)


[float]
=== Parameters
`x`: значення. Дозволені типи даних: будь-який тип. +
`h`: старший (лівий) байт слова. +
`l`: молодший (правий) байт слова.


[float]
=== Returns
Тип даних: link:../../data-types/word[`word`].
