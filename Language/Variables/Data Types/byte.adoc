= byte

// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Байт зберігає 8-бітове беззнакове число, від 0 до 255.
[%hardbreaks]


[float]
=== Syntax
`byte var = val;`


[float]
=== Parameters
`var`: ім'я змінної. +
`val`: значення, яке присвоюється цій змінній.
