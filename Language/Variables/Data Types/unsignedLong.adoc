= unsigned long

// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Змінні типу unsigned long - це змінні розширеного розміру для зберігання чисел, і вони зберігають 32 біти (4 байти). На відміну від стандартних long, unsigned long не зберігають від'ємні числа, тому їхній діапазон становить від 0 до 4 294 967 295 (2^32 - 1).
[%hardbreaks]

[float]
=== Syntax
`unsigned long var = val;`


[float]
=== Parameters
`var`: ім'я змінної. +
`val`: значення, яке ви їй призначаєте.
