= void

// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Ключове слово `void` використовується тільки в оголошеннях функцій. Воно вказує на те, що функція не повинна повертати жодної інформації до функції, з якої вона була викликана.
[%hardbreaks]

--
