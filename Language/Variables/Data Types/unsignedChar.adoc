= unsigned char

// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Беззнаковий тип даних, який займає 1 байт пам'яті. Те саме, що й тип даних `link:../byte[byte]`.

Тип даних unsigned char кодує числа від 0 до 255.

Для узгодженості стилю програмування Arduino слід віддавати перевагу типу даних link:../byte[`byte`].
[%hardbreaks]


[float]
=== Syntax
`unsigned char var = val;`


[float]
=== Parameters
`var`: ім'я змінної. +
`val`: значення, яке ви їй призначаєте.
