[float]
=== Опис
Тип short є 16-бітним типом даних.

На всіх платах Arduino (на базі ATMega та ARM) `short` зберігає 16-бітне (2-байтове) значення. Це дає діапазон від -32,768 до 32,767 (мінімальне значення -2^15 та максимальне значення (2^15) - 1).
[%hardbreaks]

[float]
=== Синтаксис
`short var = val;`


[float]
=== Параметри
`var`: ім'я змінної. +
`val`: значення, яке ви присвоюєте цій змінній.
