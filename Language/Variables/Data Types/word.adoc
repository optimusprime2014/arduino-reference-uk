= word

// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Слово (word) може зберігати ціле число без знака розміром принаймні 16 біт (від 0 до 65535).
[%hardbreaks]


[float]
=== Syntax
`word var = val;`


[float]
=== Parameters
`var`: ім'я змінної. +
`val`: значення, яке присвоюється цій змінній.
