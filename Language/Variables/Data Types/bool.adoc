= bool

// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
У тип `bool` можна записати одне з двох значень: `link:../../constants/truefalse[true]` або `link:../../constants/truefalse[false]`. (Кожна змінна `bool` займає один байт пам'яті.)


[%hardbreaks]


[float]
=== Syntax
`bool var = val;`


[float]
=== Parameters
`var`: ім'я змінної. +
`val`: значення, яке присвоюється цій змінній.

=== Example Code
// Describe what the example code is all about and add relevant code   ►►►►► THIS SECTION IS MANDATORY ◄◄◄◄◄
Цей код показує, як використовувати тип даних `bool`.

[source,arduino]
----
int LEDpin = 5;     // LED on pin 5
int switchPin = 13; // momentary switch on 13, other side connected to ground

bool running = false;

void setup() {
  pinMode(LEDpin, OUTPUT);
  pinMode(switchPin, INPUT);
  digitalWrite(switchPin, HIGH);  // turn on pullup resistor
}

void loop() {
  if (digitalRead(switchPin) == LOW) {
    // switch is pressed - pullup keeps pin high normally
    delay(100);                     // delay to debounce switch
    running = !running;             // toggle running variable
    digitalWrite(LEDpin, running);  // indicate via LED
  }
}
----
