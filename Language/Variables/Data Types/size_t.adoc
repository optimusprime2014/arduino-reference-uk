= size_t

// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
`size_t` — це тип даних, здатний представляти розмір будь-якого об’єкта в байтах. Прикладами використання `size_t` є тип значення, що повертається функціями `link:../../utilities/sizeof[sizeof()]` та `link:../../../functions/communication/serial/print[Serial.print()]`.
[%hardbreaks]


[float]
=== Синтаксис
`size_t var = val;`


[float]
=== Параметри
`var`: ім'я змінної. +
`val`: значення, яке призначається цій змінній.
