= highByte()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Витягує старший (крайній лівий) байт слова (або другий наймолодший байт більшого типу даних).
[%hardbreaks]


[float]
=== Синтаксис
`highByte(x)`


[float]
=== Параметри
`x`: значення будь-якого типу


[float]
=== Повертає
Тип даних: `byte`.
