= SPI


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис


Ця бібліотека дозволяє вам спілкуватися з пристроями SPI, при цьому Arduino виступає в ролі головного (controller) пристрою. Ця бібліотека постачається з кожною платформою Arduino (avr, megaavr, mbed, samd, sam, arc32), тому *вам не потрібно встановлювати* бібліотеку окремо.

Щоб використовувати цю бібліотеку

`#include <SPI.h>`

Щоб дізнатися більше про Arduino та SPI, ви можете відвідати посібник https://docs.arduino.cc/learn/communication/spi[Arduino & Serial Peripheral Interface (SPI)].
