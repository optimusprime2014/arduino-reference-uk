= Stream


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Stream є базовим класом для потоків на основі символів і двійкових даних. Він не викликається безпосередньо, але викликається щоразу, коли ви використовуєте функцію, яка на нього спирається.

Stream визначає функції читання в Arduino. Використовуючи будь-які основні функції, які використовують `read()` або подібний метод, ви можете сміливо припустити, що вони викликають клас Stream. Для таких функцій, як `print()`, Stream успадковується від класу Print.

Деякі з бібліотек, які спираються на Stream, включають:

* link:https://www.arduino.cc/en/Reference/serial[Serial]
* link:https://www.arduino.cc/en/Reference/wire[Wire]
* link:https://www.arduino.cc/en/Reference/Ethernet[Ethernet]
* link:https://www.arduino.cc/en/Reference/SD[SD]
