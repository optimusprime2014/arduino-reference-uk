= cos(rad)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Обчислює косинус кута (у радіанах). Результат буде між -1 та 1.
[%hardbreaks]


[float]
=== Syntax
`cos(rad)`


[float]
=== Parameters
`rad`: Кут у радіанах. Допустимі типи даних: `float`.


[float]
=== Returns
Косинус кута. Тип даних: `double`.
