= tan(rad)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Обчислює тангенс кута (у радіанах). Результат буде від мінус нескінченності до плюс нескінченності.
[%hardbreaks]


[float]
=== Syntax
`tan(rad)`


[float]
=== Parameters
`rad`: Кут у радіанах. Допустимі типи даних: `float`.


[float]
=== Returns
Тангенс кута. Тип даних: `double`.
