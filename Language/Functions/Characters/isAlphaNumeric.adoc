= isAlphaNumeric(thisChar)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Перевіряє, чи є символ буквено-цифровим (тобто літерою або цифрою). Повертає true, якщо thisChar містить або цифру, або літеру.
[%hardbreaks]


[float]
=== Синтаксис
`isAlphaNumeric(thisChar)`


[float]
=== Параметри
`thisChar`: змінна. Дозволені типи даних: `char`.


[float]
=== Повертає
`true`: якщо thisChar є буквено-цифровим.

--
// OVERVIEW SECTION ENDS



// HOW TO USE SECTION STARTS
[#howtouse]
--

[float]
=== Приклад коду

[source,arduino]
----
if (isAlphaNumeric(myChar)) { // tests if myChar isa letter or a number
  Serial.println("The character is alphanumeric");
}
else {
  Serial.println("The character is not alphanumeric");
}
----
