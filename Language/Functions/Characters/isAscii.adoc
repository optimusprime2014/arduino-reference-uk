= isAscii(thisChar)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Перевіряє, чи є символ символом ASCII. Повертає true, якщо thisChar містить символ ASCII.
[%hardbreaks]


[float]
=== Синтаксис
`isAscii(thisChar)`


[float]
=== Параметри
`thisChar`: змінна. Дозволені типи даних: `char`.


[float]
=== Повертає
`true`: якщо thisChar є символом ASCII.

--
// OVERVIEW SECTION ENDS



// HOW TO USE SECTION STARTS
[#howtouse]
--

[float]
=== Приклад коду

[source,arduino]
----
if (isAscii(myChar)) {  // tests if myChar is an Ascii character
  Serial.println("The character is Ascii");
}
else {
  Serial.println("The character is not Ascii");
}
----
