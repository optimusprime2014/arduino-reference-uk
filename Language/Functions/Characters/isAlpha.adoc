= isAlpha(thisChar)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Опис
Перевіряє, чи є символ літерою. Повертає true, якщо thisChar містить літеру.
[%hardbreaks]


[float]
=== Синтаксис
`isAlpha(thisChar)`


[float]
=== Параметри
`thisChar`: змінна. Дозволені типи даних: `char`.


[float]
=== Повертає
`true`: якщо thisChar є літерою.

--
// OVERVIEW SECTION ENDS



// HOW TO USE SECTION STARTS
[#howtouse]
--

[float]
=== Приклад коду

[source,arduino]
----
if (isAlpha(myChar)) {  // tests if myChar is a letter
  Serial.println("The character is a letter");
}
else {
  Serial.println("The character is not a letter");
}
----
