= detachInterrupt()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Вимикає задане переривання.
[%hardbreaks]


[float]
=== Syntax
- `detachInterrupt(digitalPinToInterrupt(pin))` (рекомендовано) +
- `detachInterrupt(interrupt)` (не рекомендовано) +
- `detachInterrupt(pin)` (Не рекомендовано. Крім того, це працює лише на певному наборі плат.)


[float]
=== Parameters
`interrupt`: номер переривання, яке потрібно вимкнути (дивіться link:../attachinterrupt[attachInterrupt()] для отримання додаткової інформації). +
`pin`: номер контакту Arduino переривання, яке потрібно вимкнути


[float]
=== Returns
Нічого
